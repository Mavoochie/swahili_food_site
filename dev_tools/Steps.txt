PHASE 1 — PROJECT SETUP
============================================================================================

STEP 1: Create project and initialize Git
------------------------------------------------------------
[ ] 1a. Open Git Bash and run:
            mkdir swahili_food_site
            cd swahili_food_site
            git init

[ ] 1b. Create a README:
            echo "# Swahili Food Site" > README.md

[ ] 1c. First commit:
            git add .
            git commit -m "init: create project folder and README"

------------------------------------------------------------
STEP 2: Set up Django backend
------------------------------------------------------------
[ ] 2a. Create backend folder and virtual environment:
            mkdir backend
            cd backend
            python -m venv venv
            source venv/Scripts/activate        ← Git Bash (Windows)

[ ] 2b. Install all dependencies:
            pip install django djangorestframework djangorestframework-authtoken django-cors-headers

[ ] 2c. Save dependencies to file:
            pip freeze > requirements.txt

[ ] 2d. Create Django project inside backend folder:
            django-admin startproject swahilifood .
            ⚠️ The dot (.) at the end is important — don't forget it

[ ] 2e. Go back to root folder and commit:
            cd ..
            git add .
            git commit -m "feat(backend): initialize Django project with all dependencies"

------------------------------------------------------------
STEP 3: Create Django apps
------------------------------------------------------------
[ ] 3a. Go into backend and activate venv:
            cd backend
            source venv/Scripts/activate

[ ] 3b. Create all three apps:
            python manage.py startapp users
            python manage.py startapp dishes
            python manage.py startapp comments

[ ] 3c. Open swahilifood/settings.py and find INSTALLED_APPS.
        Add these lines inside the list:
            'rest_framework',
            'rest_framework.authtoken',
            'corsheaders',
            'users',
            'dishes',
            'comments',

[ ] 3d. In the same settings.py, find MIDDLEWARE and add
        THIS LINE at the very top of the list:
            'corsheaders.middleware.CorsMiddleware',

[ ] 3e. At the bottom of settings.py add:
            CORS_ALLOW_ALL_ORIGINS = True

            AUTH_USER_MODEL = 'users.User'

            REST_FRAMEWORK = {
                'DEFAULT_AUTHENTICATION_CLASSES': [
                    'rest_framework.authentication.TokenAuthentication',
                ],
            }

[ ] 3f. Commit:
            cd ..
            git add .
            git commit -m "feat(backend): create users, dishes, comments apps and configure settings"

STEP 4: Build the User model
------------------------------------------------------------
[ ] 4a. Open backend/users/models.py and replace with:

            from django.contrib.auth.models import AbstractUser
            from django.db import models

            class User(AbstractUser):
                ROLE_CHOICES = [('admin', 'Admin'), ('user', 'User')]
                role = models.CharField(max_length=10, choices=ROLE_CHOICES, default='user')

STEP 10: Build the Dish model
------------------------------------------------------------
[ ] 10a. Open backend/dishes/models.py and replace with:

            from django.db import models

            class Dish(models.Model):
                name        = models.CharField(max_length=200)
                description = models.TextField()
                price       = models.DecimalField(max_digits=10, decimal_places=2)
                created_at  = models.DateTimeField(auto_now_add=True)

------------------------------------------------------------
STEP 16: Configure Vite proxy
------------------------------------------------------------
[ ] 16a. Open frontend/vite.config.js and replace with:

            import { defineConfig } from 'vite'
            import react from '@vitejs/plugin-react'

            export default defineConfig({
              plugins: [react()],
              server: {
                proxy: {
                  '/api': 'http://localhost:8000',
                }
              }
            })

         ⚠️ This is what lets React talk to Django
         Without this, API calls will fail with CORS errors

[ ] 16b. Commit:
            git add .
            git commit -m "config(frontend): add Vite proxy to forward /api requests to Django on port 8000"
